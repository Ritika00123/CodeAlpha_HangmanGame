# Hangman Game

## Description
A simple text-based Hangman game developed in Python. The player guesses a hidden word one letter at a time. The game ends when the player guesses the word correctly or runs out of attempts.

## Features
- Random word selection
- 6 incorrect guess limit
- User-friendly console interface
- Input validation

## Technologies Used
- Python 3

## How to Run
1. Install Python 3.
2. Download the project files.
3. Open terminal or command prompt.
4. Run:

python HangmanGame.py
